Task-02: Image Generation with Pre-trained Models

This project is part of my Generative AI Internship at Prodigy InfoTech.

Project Overview

The objective of this task is to generate images from text prompts using pre-trained generative AI models. The project demonstrates how modern AI models can transform natural language descriptions into visual images.

Using Stable Diffusion, the system takes a text prompt as input and generates a corresponding image.

Features

Generate images from text descriptions

Uses a pre-trained generative AI model

Fast image generation with optimized inference steps

Simple and easy-to-understand implementation

Technologies Used

Python

Diffusers

Transformers

PyTorch

Google Colab

Project Workflow

Install required libraries.

Load the pre-trained Stable Diffusion model.

Provide a text prompt.

Generate the image.

Save and display the generated output.

Example Prompt
"A futuristic city with flying cars at sunset"
Output

The model generates an image based on the given prompt using the Stable Diffusion pipeline.
